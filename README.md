## Tema

Projeto de uma **Consultoria em Desenvolvimento** em ASP.NET Core MVC.

## Funcionalidades

- Tela inicial com cards dos problemas/serviços.
- Exibição de nome, descrição e tempo de resposta.
- Alerta ao solicitar o serviço.
- Formulário para o cliente informar nome e e-mail.
- Painel do Tech Líder para aprovar ou reprovar solicitações.
- Encaminhamento para WhatsApp do dev disponível.
- Autenticação com **Claims** e **Cookies**.
- Layout ajustado para verificar login.
- Estilização com **Bootstrap**.
